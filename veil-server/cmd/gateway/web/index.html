Veil

E2EE мессенджер. Вся крипто в Rust — ключи не пересекают FFI-границу, не оседают в JS heap, не видны серверу. Никогда.

Возможности
Построен правильно с нуля
Переписан после того, как предыдущий вырос до 22k LOC монолита с криптой на TweetNaCl в JS.
🔐

End-to-end шифрование

X3DH для установки сессии, Double Ratchet для forward secrecy, Sender Keys для больших групп. Всё как у Signal — только в Rust.

XChaCha20-Poly1305
🦀

Крипто только в Rust

Ни одной криптографической операции в JS или TypeScript. UI получает plaintext и метаданные — больше ничего. mlock() для страниц с ключами, zeroize on drop.

native FFI
📱

Desktop + Mobile

Tauri v2 для десктопа, React Native для мобильного. Единое ядро на Rust через UniFFI bidings для Kotlin и Swift.

Tauri · Expo
🔍

Локальный поиск

Tantivy индекс живёт только на устройстве. Поисковые запросы не уходят на сервер. Cmd-K палитра для быстрой навигации.

Tantivy 0.22
📁

Файлы до 2 ГБ

tus.io resumable загрузки. Файл шифруется до отправки — chunked AEAD с привязкой каждого чанка к индексу и финальному флагу. Сервер хранит opaque blob.

chunked AEAD
🔔

Push без FCM

UnifiedPush + self-hosted ntfy. Payload зашифрован, заполнен до 2 КБ, с jitter задержкой. Ntfy-оператор видит только размер и тайминг.

UnifiedPush · ntfy
🗂️

Серверы и каналы

Discord-стиль: серверы, каналы, роли, инвайты. Группы с Sender Keys. Зашифрованы так же как и DM — сервер не видит контент.

Sender Keys
🔗

Secure Share

Ссылки с TTL и счётчиком просмотров. Расшифровка происходит прямо в браузере через минимальный WASM-модуль (~50 KB). Сервер не видит ключ.

WASM · Argon2id
